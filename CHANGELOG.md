# Changelog

Todos los cambios notables en este proyecto serán documentados en este archivo.

El formato está basado en [Keep a Changelog](https://keepachangelog.com/es-ES/1.0.0/),
y este proyecto adhiere a [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Estructura de proyecto organizada y documentada
- Guías de instalación y despliegue
- Templates de issues y pull requests
- CI/CD con GitHub Actions
- Configuración para Vercel

### Changed
- Mejorada la estructura de carpetas según mejores prácticas
- Documentación reorganizada en carpeta `docs/`
- README mejorado con badges y estructura profesional

## [0.1.0] - 2024-11-15

### Added
- Catálogo de productos con filtros
- Carrito de compras con estado persistente
- Checkout vía WhatsApp
- Autenticación con Supabase
- Panel de administración completo
- Gestión de inventario
- Sistema de precios personalizados por cliente
- Sistema de descuentos
- Gestión de categorías y productos
- Gestión de pedidos con historial
- Importación/Exportación de datos
- Favoritos (anteriormente "Lista de deseos")
- Testing con Vitest y Playwright
- Monitoreo con Sentry
