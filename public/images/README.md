# Logo de MaterialesYA

Coloca aquí el archivo del logo de MaterialesYA.

## Archivo requerido:
- `logo.materiales.jpeg` - Logo principal de MaterialesYA

## Especificaciones recomendadas:
- Formato: JPEG
- Tamaño: Mínimo 200x200px (se escalará automáticamente)
- Fondo: Transparente o con fondo naranja (#FF6600)

## Ubicación del archivo:
```
public/images/logo.materiales.jpeg
```

El componente `Logo` buscará este archivo automáticamente. Si no existe, mostrará un SVG de fallback basado en el diseño del logo hexagonal.
